# SSH

## Generate SSH key

```bash
ssh-keygen -t ed25519 -C "your_email@example.com"
```

## Add SSH key to ssh-agent

```bash
eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_ed25519
```

## Add SSH key to GitHub

- Settings > SSH and GPG keys > New SSH > Add SSH key

```bash
cat ~/.ssh/id_ed25519.pub
```
