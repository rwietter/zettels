# CSS Namespaces

Lista de CSS Namespaces

- **.l-** : layouts
- **.o-** : objects
- **.c-** : components
- **.js** : JavaScript hooks
- **.is** |.has- : state classes
- **.t1** |.s1 : typography sizes
- **.u-** : utility classes
