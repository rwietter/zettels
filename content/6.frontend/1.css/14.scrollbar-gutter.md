# Scrollbar

A propriedade `scrollbar-gutter` permite que reservemos o espaço do _scrollbar_ mesmo que ele não esteja na tela, isso impede shifts de conteúdo de quando o _scrollbar_ aparece.

```css
html {
 scrollbar-gutter: stable;
}
```
